# Backend API — Dónde Comemos Hoy

Este directorio contiene la API REST de **Dónde Comemos Hoy**, una aplicación full-stack orientada a ayudar a familias a encontrar restaurantes cómodos para ir con niños.

> Para ver el proyecto completo, consulta el README principal del repositorio.

## Funcionalidades principales

- Gestión de restaurantes
- Filtros familiares
- Sistema de sugerencias
- Aprobación y rechazo desde dashboard
- Autenticación JWT para administración
- Favoritos globales
- Geocodificación y mapas
- Documentación Swagger

## Stack

- Node.js
- Express
- Prisma ORM
- PostgreSQL
- JWT
- Swagger
- Winston

cd backend
